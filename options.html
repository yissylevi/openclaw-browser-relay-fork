Getting started

If you see a red ! badge on the extension icon, the relay server is not reachable. Start OpenClaw’s browser relay on this machine (Gateway or node host), then click the toolbar button again.

Click the extension icon to choose Coding or Research for the active tab. Badge meanings: ON = attached · … = attaching · ! = error.

Full guide (install, remote Gateway, security): docs.openclaw.ai/tools/chrome-extension

Relay connection

Sets the default relay for tabs without per-tab overrides.

Default port: 18792. Extension connects to: http://127.0.0.1:<port>/. Gateway token must match gateway.auth.token (or OPENCLAW_GATEWAY_TOKEN).

Per-tab relay assignment

Use this when running multiple relays/agents. Assign the current tab to a specific relay port/token.

Tip: open a tab, click it, then assign it here to force Coding vs Research.

Assignment is stored by tab id. Closing the tab clears the override.

Chat browser auto-open

Open the OpenClaw chat browsers on Chrome startup. Each URL opens in its own window.

Default base ports: 18789 and 19001. Effective URLs become http://127.0.0.1:<base>/__openclaw__/canvas/. If you provide explicit URLs above, the base ports are ignored.
